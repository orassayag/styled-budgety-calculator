# Node Vidly Deployment

Built in June 2018.
Styled budgety calculator to calculate budget with positive and negative items.
Built using HTML + CSS / Pure JavaScript ES6.

## Getting Started

Clone the project to your computer.

### Prerequisites

Just run the project with Chrome.
